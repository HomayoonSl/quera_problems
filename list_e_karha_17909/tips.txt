چه چیز‌هایی از حل این سوال می‌توان آموخت؟
    
    ۱. برای گرفتن flag یا function از ترمینال می‌توان از case استفاده کرد
    
    ۲. اگر بخواهیم بدانیم که رشته‌ي ورودی null است، می‌توان از z- در if استفاده کرد.(یعنی اگر پوچ بود، داخل if شو!) به این صورت 
    if [[ -z $1 ]]; then
        something...
    fi
    ۳. مشابه ۲ اگر بخواهیم بدانیم که رشته null نیست از n- استفاده می‌کنیم. 
    
    ۴.اگر بخواهیم چک کنیم که داخل text رشته‌ای داریم یا نه از s- استفاده می‌کنیم. این flag اگر فایل text مقدار داشته باشد، عمل می‌کند. (خط ۳۵)
    
    ۵. دستور awk واقعا دستور کار راه‌انداز خوبی‌ست. به خط ۳۶ نگاه کنید که چگونه text را تغییر می‌دهد و چاپ می‌کند.
    
    ۶. و نکات دیگر که یادم رفت در زمان حل سوال یادداشت کنم و الان به چشمم نمی‌آید.
